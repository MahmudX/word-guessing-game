# Word Guessing Game
Simple word guessing game written in golang.

```go
successTexts := []string{

  "Correct! ✅",

  "LEGIT 👏🏻",
  "You are correct! 👍🏻",
  "Damn you are GOOD! 👏🏻👏🏻",

```
